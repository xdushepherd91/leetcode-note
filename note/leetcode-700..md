### 题目

[[leetcode 700.]](https://leetcode-cn.com/problems/search-in-a-binary-search-tree/submissions/) 二叉搜索树中的搜索

### 描述

给定二叉搜索树（BST）的根节点和一个值。 你需要在BST中找到节点值等于给定值的节点。 返回以该节点为根的子树。 如果节点不存在，则返回 NULL。

例如，

```
给定二叉搜索树:

        4
       / \
      2   7
     / \
    1   3

和值: 2

你应该返回如下子树:

      2     
     / \   
    1   3
```
